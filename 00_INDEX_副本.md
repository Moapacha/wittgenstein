# Polyglot / Modality Harness — Final Delivery Index

This folder contains the final delivery package for the hackathon project.

## Files

1. `01_Build_Book.md`
   - Main human/developer document.
   - Detailed product positioning, architecture, technical paths, work plan, CLI/skill design, evaluation, and execution guidance.

2. `02_AI_Execution_Context.md`
   - Compact, fixed context for Claude Code / Codex / Cursor / OpenCode.
   - Intended to be pasted into a fresh session or used as `AGENTS.md` + repo docs seed.

3. `03_Research_Reference_Dossier.md`
   - Annotated reference pack.
   - Includes official articles, papers, repos, and why each matters.

4. `04_Pitch_Pack.md`
   - Pitch content, logic flow, demo choreography, story arc, CTA lines, objection handling, and slide-by-slide copy.

5. `05_landing_demo.html`
   - Self-contained landing page demo.
   - Open locally in a browser.

## Recommended usage order

1. Read `01_Build_Book.md` first.
2. Hand `02_AI_Execution_Context.md` to coding agents.
3. Use `03_Research_Reference_Dossier.md` when you need proof, references, or rebuttals.
4. Use `04_Pitch_Pack.md` to rehearse and cut your final talk.
5. Open `05_landing_demo.html` for visual packaging and demo page content.
