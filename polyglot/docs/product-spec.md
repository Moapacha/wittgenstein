# Product Spec

## This Phase Ships

- pnpm TypeScript monorepo
- shared schema package
- harness runtime with config, routing, telemetry, manifests, retry, seed, budget
- four codec packages with typed skeletons
- `polyglot` CLI
- official site scaffold in the same repo
- docs, ADRs, examples, benchmarks scaffold, fixtures, CI

## This Phase Does Not Ship

- real image adapter or decoder wiring
- live XTTS, CLAP, synth, HyperFrames, or sensor generators
- site marketing copy
- benchmark results
- committed secrets or weights
